# Stock Report Expression Guide — 样例报告表达指南

## 1. 表达目标

报告要像研究员写出的完整成品，而不是数据库导出的说明。表达应做到：

```text
有主线、有分歧、有证据、有模型、有节奏、有反证。
```

但不能牺牲证据纪律。语言可以有判断力，但每个判断必须能追溯。

## 2. 段落写法

### 2.1 好段落结构

```text
结论句 → 证据/数据 → 原因解释 → 投研含义 → 风险或边界
```

示例模板：

```text
{{业务}} 是公司当前最重要的 {{现金牛/增长引擎/期权业务}}。
{{period}} 该业务实现收入 {{value}}，占比 {{pct}}，毛利率 {{gm}}。
这一变化主要来自 {{driver_1}} 和 {{driver_2}}。
从投研角度看，关键不在于 {{表象}}，而在于 {{核心变量}}。
但需要注意，{{risk_or_missing_data}} 仍未被完全验证。
```

### 2.2 前言写法

前言要压缩全篇逻辑，不要泛泛介绍。

```text
{{company_name}} 是一家 {{传统身份}}，但当前正在被 {{新变量}} 重新定价。
市场目前主要争议在 {{争议1}} 和 {{争议2}}。
本文认为，真正决定未来 12-24 个月股价/估值重估的变量是 {{核心变量}}。
```

## 3. 样例报告的可迁移表达能力

### 3.1 “表观利润失真”表达

用于非经常性损益、公允价值变动、股权激励、并表等场景。

```text
表观看，公司 {{metric}} 下滑；但穿透后，{{adjusted_metric}} 反而改善。
所谓“业绩下滑”本质上是 {{accounting_item}} 的扰动，而不是主营业务出了系统性问题。
```

### 3.2 “业务角色”表达

```text
{{business}} 是公司的“现金牛”。
{{business}} 是公司的“增长引擎”。
{{business}} 是公司的“估值期权”。
{{business}} 是公司的“收入压舱石但利润贡献有限”。
```

### 3.3 “行业主线”表达

```text
该行业当前处在 {{阶段}}：{{需求变量}} 与 {{供给变量}} 同时变化。
对公司而言，最重要的不是行业总空间，而是 {{company_specific_exposure}}。
```

### 3.4 “盈利预测”表达

```text
盈利预测本质上是对 {{变量1}}、{{变量2}}、{{变量3}} 的概率加权判断。
我们将未来三年分为 {{阶段1}}、{{阶段2}}、{{阶段3}}。
最敏感变量是 {{sensitive_variable}}，每变化 {{delta}}，对归母净利润影响约 {{impact}}。
```

### 3.5 “估值争议”表达

```text
当前估值的核心争议不是静态 PE 高低，而是市场是否已经 price in {{future_driver}}。
若 {{assumption}} 成立，当前估值仍有解释空间；若 {{counter_assumption}} 成立，则估值存在压缩风险。
```

## 4. 禁止表达

```text
- “确定会涨 / 必然重估 / 强烈买入 / 立即建仓”。
- 未写来源的目标价、仓位、短线操作策略。
- 把券商预测写成公司事实。
- 把新闻传闻写成公告事实。
- 把互动易回复写成已实现收入。
- 把结构化财务指标直接归因到某个细分业务。
```

## 5. 推荐替代表达

| 不推荐 | 推荐 |
|---|---|
| 买入 | high_conviction_watch，附条件和证伪指标 |
| 卖出 | avoid_until_evidence 或 risk_watch |
| 目标价 | 场景估值区间，仅作为模型输出 |
| 仓位建议 | 跟踪优先级 / 观察清单状态 |
| 短线操作 | 事件窗口 / 技术状态观察 |

## 6. TODO 的表达

缺证据不要隐藏，应该写入正文：

```text
目前尚未找到公司直接披露 {{missing_item}} 的证据。
已检索 {{sources_checked}}，结论暂列为 TODO。
后续需要通过 {{next_sources}} 验证。
```

## 7. 结论页模板

```text
综合来看，{{company_name}} 当前的研究价值来自 {{driver_1}}、{{driver_2}} 和 {{driver_3}}。
其中，已被证据支持的是 {{fact_supported_points}}；仍需验证的是 {{open_questions}}。
如果 {{positive_condition}}，公司有望进入 {{positive_scenario}}；如果 {{negative_condition}}，本文判断将被削弱。
因此，本报告将其列为 {{research_status}}，后续重点跟踪 {{watch_items}}。
```
