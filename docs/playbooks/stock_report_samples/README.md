# 个股研究报告样例库

> 本目录用于保存“个股研究报告写作标准”样例，不是证据库、正式报告库或投资建议库。

## 使用边界

- 可以学习：章节组织、业务拆分颗粒度、行业映射、估值讨论框架、风险提示位置。
- 不可以当作证据：不登记 `evidence_id`、`claim_id`、`metric_id`，不得支撑正式结论。
- 不可以复用交易建议：样例中出现的评级、目标价、仓位、买入/卖出/持有等表达，只能作为需要改写或规避的语言样本。
- 正式项目产出应改写为：观察结论、场景假设、风险清单、待验证问题、证据缺口。
