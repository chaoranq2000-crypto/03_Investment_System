# R5 Sample Reports Benchmark Policy

This directory is for benchmark metadata only.

Sample reports are used only for structure, density, section capability, and evidence-requirement benchmarking.

Do not paste full external brokerage reports, paid reports, copyrighted articles, or copied long-form sample text into this repository. R5 may learn from sample structure, section density, issue types, and source-gap handling, but it must not copy proprietary wording or trading-advice language.

Allowed benchmark records:

- report title, source category, date, and internal note about structure;
- section-density observations;
- anonymized checklist items;
- no-advice boundary notes;
- TODO/source-gap handling examples written in original repo wording.

Disallowed:

- full report text;
- long copied excerpts;
- copied ratings, target prices, position guidance, or action instructions;
- using sample report facts as evidence for an A-share company.
- copying sample ratings, target prices, trading advice, or position guidance into system outputs.
