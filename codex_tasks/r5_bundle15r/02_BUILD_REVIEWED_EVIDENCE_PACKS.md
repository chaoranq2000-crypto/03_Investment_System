# 15R-2 — Build reviewed official evidence packs

## Actions

1. Generate one visibly incomplete scaffold per Bundle 14R case.
2. Replace scaffold placeholders only with reviewed, physically archived
   official evidence.
3. Record reviewer, timestamp, source class, publication date, covered period,
   archive path, SHA-256, locator, and limitations.
4. Map each record to an exact Bundle 14R driver and one or more research
   questions.
5. Record units, periods, definitions, confidence, overlap rules, and stale
   triggers.
6. Preserve blocked or context-only records when disclosure is insufficient.
7. Never use the four narrative samples, generated Readers, or model outputs as
   evidence.

## Exit criteria

Every critical question is classified as confirmed, bounded estimate,
context-only, blocked, or not applicable. No unsupported numeric value is added.
