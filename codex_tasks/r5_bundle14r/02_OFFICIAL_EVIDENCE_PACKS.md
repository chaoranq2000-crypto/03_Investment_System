# 14R-2 — Build reviewed official evidence packs

## Scope

Create one evidence pack per golden case. Samples provide dimensions only and cannot be cited as evidence.

## Required record fields

- claim/metric identifier;
- source class and source identifier;
- URL or repository file;
- publication date and covered period;
- unit, currency, and business definition;
- fact/management statement/estimate/inference label;
- confidence and reviewer status;
- overlap rule;
- dependent driver/model/report nodes;
- stale trigger.

## Exit criteria

Every required research question is classified as official fact, bounded estimate, context only, or blocked. Missing information remains visible and generates an owner-assigned backflow item.
