# 14R-5 — Semantic and adversarial quality gate

Run the non-compensating gate and adversarial corpus. Mandatory failures include:

- long but generic narrative;
- repeated thesis across chapters;
- non-empty causal fields without company-specific variables;
- forecast without driver-to-statement bridge;
- low-confidence peers treated as qualified;
- future events without dates, operating transmission, or falsification;
- past events presented as future catalysts;
- sample text promoted to evidence;
- issuer names/tickers embedded in generic runtime code;
- an ineligible valuation method used despite a high aggregate score.

Exit only when every core gate passes. Aggregate score cannot compensate for a failed core gate.
