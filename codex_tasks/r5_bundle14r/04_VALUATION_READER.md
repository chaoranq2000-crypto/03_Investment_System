# 14R-4 — Valuation qualification and Reader generation

## Valuation

Qualify each method independently. Peer multiples require at least three compatible peers; DCF requires qualified FCF/WACC/terminal inputs; SOTP requires independent segment economics and eliminations; reverse valuation requires a reconciled market-value and forecast definition.

## Reader

Use the common Writer and report planner. The top three case-specific contradictions must receive at least 60% of analytical emphasis. Each core section must add a new metric, model link, conclusion, or falsification condition.

## Exit criteria

- at least one valuation method qualifies, or the case remains explicitly blocked;
- no ineligible method is used to derive a conclusion;
- no issuer-specific branch is introduced in generic runtime code;
- the report contains no direct trading/position instruction.
